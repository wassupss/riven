Freely dockable panels자유 도킹 패널

Persistent terminals영속 터미널

Monaco + multi-language LSPMonaco + 멀티 언어 LSP

Git built inGit 통합

Stage, commit, push and pull, browse diffs and read inline blame — a full Git panel without leaving the editor.스테이지·커밋·push/pull·diff·인라인 blame까지 — 에디터를 벗어나지 않는 완전한 Git 패널.

Inline edit review인라인 편집 리뷰
